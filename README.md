# 🎨 維度驅動多風格 JSON 網站系統

一個革命性的**維度驅動架構**，透過 4 個設計維度的系統性組合，從單一 JSON 數據創造 16 種完全不同的專業級視覺風格。完美的內容與設計分離解決方案，適合企業官網、內容網站、讀書會平台。

## ✨ 核心創新

### 🏗️ **維度驅動架構**
**四維設計系統** - 透過數學組合產生 2⁴ = **16 種獨特風格**：

| 維度 | 選項 | 管理範圍 |
|------|------|----------|
| **Colors** | 明亮/深色 | 背景、文字、邊框顏色系統 |
| **Typography** | 正式/輕鬆 | 字體族、粗細、行高、字距 |
| **Spacing** | 緊湊/寬鬆 | padding、margin、元素間距 |
| **Effects** | 平面/立體 | 陰影、漸層、動畫特效 |

### 🎯 **完美分離架構**
- **單一數據源**: `data/content.json` 驅動所有 16 種風格
- **零程式碼修改**: 新增風格只需一個配置檔案
- **OCP 合規**: 開放擴展，封閉修改的軟體工程原則
- **企業就緒**: 生產級架構，適合大型專案

### 🎨 **專業設計品質**
- **獲獎級標準**: 每個風格達到 Awwwards 參賽水準
- **精心調配**: 限制 3-4 種色彩，確保視覺和諧
- **高對比度**: 4.5:1+ 可讀性標準，符合 WCAG 規範
- **專業動畫**: 自定義 cubic-bezier 緩動函數和微互動

### 🔧 **開發者友善**
- **智慧路徑系統**: 自動偵測相對路徑，支援任何部署環境
- **響應式優先**: 使用 `clamp()` 函數實現流暢適配
- **模組化架構**: 每個維度管理正交的 CSS 屬性組，零衝突
- **即時熱更新**: 修改 JSON 即時看到所有風格變化

## 🔧 開發指南

### 新增維度風格 - 簡單 3 步驟

**步驟 1**: 建立風格配置檔案

```javascript
// js/styles/your-style.js
import { Dark, Casual, Loose, Flat } from '../dimensions/';

export default {
  id: 'your-style',
  name: 'Your Amazing Style',
  description: '風格描述',
  icon: '✨',
  category: 'modern',
  dimensions: [Dark, Casual, Loose, Flat], // 四維組合定義風格
  url: 'pages/your-style.html',
  buttonText: '體驗風格 →',

  // 可選：客製化樣式覆蓋
  cardStyle: {
    background: 'bg-gradient-to-br from-gray-900 to-black',
    text: 'text-white',
    hover: 'hover:shadow-2xl hover:scale-105'
  }
};
```

**步驟 2**: 註冊到系統

```javascript
// js/styles/index.js
import yourStyle from './your-style.js';

const styles = [
  zenStyle,
  luxuryStyle,
  yourStyle  // 加入新風格
];
```

**步驟 3**: 建立專用頁面 (可選)

```html
<!-- pages/your-style.html -->
<!DOCTYPE html>
<html>
<!-- 使用專用載入器獲得最佳體驗 -->
<script type="module" src="../js/loaders/your-style.js"></script>
</html>
```

### 系統會自動處理：
- ✅ 從四個維度自動生成 CSS 變數和類名
- ✅ 註冊到風格註冊器和路由系統
- ✅ 在主頁面顯示風格卡片和預覽
- ✅ 提供完整的響應式和互動體驗
- ✅ 智慧載入內容並套用風格樣式

### 維度系統詳解

每個維度都是獨立的 CSS 屬性組，確保零衝突：

```javascript
// 四個維度的組合創造無限可能
const dimensions = {
  Colors: [Light, Dark],        // 2 種選擇
  Typography: [Formal, Casual], // 2 種選擇
  Spacing: [Compact, Loose],    // 2 種選擇
  Effects: [Flat, Dimensional]  // 2 種選擇
};

// 總共 2×2×2×2 = 16 種獨特組合
// 目前已實現 10 種，還有 6 種等待創造！
```
