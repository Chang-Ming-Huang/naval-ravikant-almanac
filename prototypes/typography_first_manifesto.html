Serif Typography
Elegance

襯線字體承載著數百年的印刷傳統，每一個筆觸都充滿了人文氣息。 它們在正文閱讀中提供最佳的視覺流暢性，是文學與學術的理想選擇。

Sans-Serif Typography
MODERN

無襯線字體體現了現代主義的純粹美學，簡潔而有力。 它們在數位媒體中表現卓越，適合標題和界面設計， 代表著當代設計的核心精神。

Monospace Typography
CODE

等寬字體是程式設計師的語言，每個字元佔據相同的空間， 創造了完美的對齊和節奏感。它們代表了邏輯、精確和技術美學。

Variable Font Weights
WEIGHT

可變字重系統讓設計師能夠精確控制文字的視覺重量， 從最輕盈的 100 到最厚重的 900，每一級都承載著不同的情感。

Letter Spacing
SPACE

字母間距是隱形的設計師，它控制著文字的呼吸節奏。 緊密的間距創造親密感，寬鬆的間距則帶來優雅和空氣感。

Line Height Control
LINE
HEIGHT

行高決定了文字的垂直節奏，影響著讀者的視覺舒適度。 它是版面設計中最重要的呼吸空間，決定了內容的可讀性和美感。
