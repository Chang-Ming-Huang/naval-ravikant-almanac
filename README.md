# 🎨 多風格 JSON 網站系統 + AI 創意實驗室

一個革命性的三重網站開發系統：**維度驅動架構**、**AI創意實驗室** 和 **3D空間導航**，創造無限多種專業級視覺風格和沉浸式3D體驗。

## ✨ 核心創新

### 🏗️ **維度驅動系統** 
**四維設計系統** - 透過組合產生 2⁴ = **16 種獨特風格**：

| 維度 | 選項 | 管理範圍 |
|------|------|----------|
| **Colors** | 明亮/深色 | 背景、文字、邊框 |
| **Typography** | 正式/輕鬆 | 字體、粗細、行高 |
| **Spacing** | 緊湊/寬鬆 | 間距、布局 |
| **Effects** | 平面/立體 | 陰影、動畫、特效 |

**內容與設計完全分離** - 單一 `data/content.json` 驅動所有風格

### 🎨 **AI 創意實驗室**
**完全獨立的HTML藝術作品** - 30個專業級風格頁面：
- 🏗️ **建築與空間系列**: Nordic Zen, Desert Modernism, Swiss Precision, Glass Architecture, Concrete Brutalism
- 🌃 **都市與科技系列**: Tokyo Neon, Fintech Modern, Neuromorphic Reality, Digital Mysticism  
- 🏥 **專業服務系列**: Medical Professional, Luxury Hospitality, Corporate Elite, Luxury Service
- 🎭 **文化藝術系列**: Artisan Craft, Museum Gallery, Publishing House, Concert Hall
- 💎 **精品創意系列**: 16個其他風格涵蓋各種創新設計理念
- 🧠 **智能偏好學習**: 用戶刪除風格會自動學習偏好避免重複

## 🔧 開發指南

### 維度驅動系統 - 新增風格

```javascript
// js/styles/your-style.js
import { Dark, Casual, Loose, Flat } from '../dimensions/';

export default {
  id: 'your-style',
  name: 'Your Amazing Style',
  dimensions: [Dark, Casual, Loose, Flat], // 四維組合定義風格
  // 其他配置...
};
```

就是這麼簡單！系統會自動：
- ✅ 生成 CSS 變數和類名
- ✅ 註冊到風格系統
- ✅ 在主頁面顯示風格卡片
